# RazorPay-Website-Clone
A responsive front-end clone of the Razorpay website built using HTML, CSS, and JavaScript. This project focuses on recreating the UI, layout, and hover interactions of the original site. JavaScript is used for basic functionality such as redirecting users to Login and Signup pages.  Tech Stack: HTML, CSS, JavaScript
